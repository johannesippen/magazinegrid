MagazineGrid Framework

Tell Your Story - in the Best Layout

MagazineGrid is a free CSS framework, designed to create amazing story layouts for the web.

Create an amazing article like on Medium, Exposure or Storehouse. Create a complete iPad magazine like the Aside Mag. Create a product landing page or a portfolio. Whatever story you want to tell, MagazineGrid helps you to set it right.

MagazineGrid comes without the clutter. It concentrates on what it does best: provide a good grid. Load more elements like buttons, forms, tables etc. as extensions.

Made for the iPad

MagazineGrid works best on the iPad in portrait and landscape orientation. The simple 8 column grid uses the full display without leaving unnecessary margins.

Phone & Desktop

No iPad? No Problem! MagazineGrid is responsive to work on bigger displays as well as on iPhone, Android or Windows Phone.

Lighter than 4kB

MagazineGrid is superlight to load fast on mobile devices. How we do it? Easy: No needless elements, no jurassic browser support.

More Semantics,
Less Divs

MagazineGrid uses semantic HTML5-elements to structure your magazine page.

An article-element wraps up your page, sections define the portions of content. Inside the sections you define which elements should match the grid and how wide they are.

Check out the following sample code for a typical story page:

<article>
  <section class="full">
    <div class="opener">
      <h1>…</h1>
      <p>…</p>
    </div>
  </section>
  <section>
    <div class="story">
      <p>…</p>
    </div>
  </section>
  <section>
    <div class="grid x2 indent1"><img src="…"></div>
    <div class="grid x2"><img src="…"></div>
    <div class="grid x2"><img src="…"></div>
  </section>
  <section>
    <div class="grid x4 indent1"><img src="…"></div>
    <div class="grid x2"><img src="…"></div>
  </section>
  <section>
    <div class="story">
      <h2>…</h2>
      <p>…</p>
      <hr>
      <blockquote>…</blockquote>
      <p>…</p>
    </div>
  </section>
  <section>
    <div class="grid x6 indent1"><img src="…"></div>
  </section>
  <section class="full">
    <img src="…">
  </section>
  <section>
    <div class="grid x3 indent1"><img src="…"></div>
    <div class="grid x3"><img src="…"></div>
  </section>
  <section>
    <div class="grid x4"><img src="…"></div>
    <div class="grid x4"><img src="…"></div>
  </section>
  <section>
    <div class="story column2 border">
      <p>…</p>
    </div>
  </section>
  <section>
    <div class="grid x3 indent1"><img src="…"></div>
    <div class="grid x3"><img src="…"></div>
  </section>

    </div>
  </section>
  <section>
    <div class="grid x4"><img src="…"></div>
    <div class="grid x4"><img src="…"></div>
  </section>
</article>
